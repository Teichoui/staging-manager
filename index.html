Ignore List

Torrents matching any entry here are never synced. Plain text (no * or ?) matches anywhere in the torrent name, ignoring dots/dashes/underscores (e.g. "Jack Ryan" matches "Jack.Ryan.S01E01"). Adding * or ? switches that entry to an exact, full-name glob against the raw name — no separator normalization, so use the actual dots/spaces and wrap with * for a "contains" match (e.g. "*Jack.Ryan*"). Brackets like [ ] always match literally, never as a character class.
